WEB

www(world wide web)라고 불리우는 웹(web)은 인터넷, 컴퓨터, 웹 브라우저(web brower)를 통해 글자, 이미지, 동영상, 음성, 음악 등의 데이터를 사용자에게 제공하거나, 사용자와 컴퓨터 또는 사용자 상호 간에 소통하게 해주는 서비스를 말합니다. 이때 웹브라우저는 서버에서 제공되는 데이터를 분석하여 화면에 보여주는 역할을 합니다. 국내 및 해외에서 널리 사용되는 웹브라우저는 마이크로스프트의 인터넷 익스플로러(internet explorer), 구글 크롬(chrome), 애플의 사파리(safari), 모질라재단 모질라(mozilla foundation)의 파이어폭스(firefox) 등이 대표적입니다.

웹(web)의 개념을 이해할 때 중요한 것이 웹브라우저 입니다. 사용자 입장에서 웹의 모든것은 웹브라우저를 통해 일어납니다. 웹브라우저를 컴퓨터에서 열고 게임을 한다던지 PC, 스마트폰과 같은 모바일 기기라던지 그 게임을 웹게임이라고 합니다. 또한 웹 메일은 사용자가 웹브라우저를 통해 메일 서비스를 제공하는 웹사이트에 접속하여 메일을 이용하기 때문에 웹 메일이라고 합니다.

웹(web)과 관련된 직업에는 여러 직업이 있지만 대표적인 웹(web) 관련 직업은 웹 디자이너, 웹 프로그래머가 있습니다. 먼저 웹 디자이너의 웹 디자인은 웹 브라우저에 들어가는 웹 페이지와 그 안의 콘텐츠, 텍스트, 이미지, 동영상 등을 디자인하는것을 말하고 웹 디지이너가 되려면 포토샵이나 일러스트레이터 같은 기본적인 디자인 툴과 웹 페이지를 제작하는 데 필요한 html, css, javacript 같은 언어 등도 다룰 줄 알아야 합니다.

웹 프로그래머의 역할인 웹 프로그래밍은 웹 브라우저에서 사용자의 마우스, 키보드 조작에 반등하는 기능, 프로그램에 필요한 데이터 저장, 수정, 삭제 등의 기능을 구현하는 것을 말합니다. 예를 들어 홈페이지 회원가입, 로그인, 게시판, 쇼핑몰의 장바구니, 배송 조회, 예약, 예매, 카드 결제, 핸드폰 결제, 실시간 이체 등과 같은 기능을 사용할 수 있겠끔 하는 역할이 웹 프로그래머의 역할입니다. 웹 프로그래머는 html, css, php, jsp, asp, 자바스크립트(javascript), (제이쿼리)jQuery 등과 데이터를 저장, 수정, 삭제를 할 수 있는 데이터베이스에 관한 충분한 지식이 있어야 합니다.
